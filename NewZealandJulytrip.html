7月6-11日新版行程

7/6抵达基督城，住基督城

安排：两人航班不同，分开接机或酒店会合。住 Rydges Latimer Christchurch，约 ¥971/人/晚。

可选：早到可去大教堂广场、基督城植物园或国际南极中心，不安排长途赶路。

7/7基督城 -> 大教堂广场 -> 好牧羊人教堂 -> 特卡波

景点：大教堂广场、Fairlie休息、特卡波湖、好牧羊人教堂。晚上住 Peppers Bluewater Resort。

可选：天气好安排特卡波观星。

7/8特卡波湖 -> 好牧羊人教堂 -> 普卡基湖 -> 库克山

景点：约翰山、普卡基湖、库克山国家公园、Tasman Lake 或 Kea Point 轻徒步/观景。

吃/休息：Astro Café 或特卡波镇午餐；晚上继续住 Peppers Bluewater Resort，天气好可安排观星。

7/9库克山雪山 -> 皇后镇

景点：库克山国家公园、Tasman Lake 或 Kea Point 轻徒步/观景；午后出发去皇后镇。

吃/休息：Old Mountaineers Café 或 Twizel/Omarama 简餐；晚住皇后镇，酒店在 Bella Vista / Sudima Five Mile / TOP 10 中三选一。

7/10米尔福德峡湾，分 A/B 线

A线：坐长途巴士/包车走瓦卡蒂普湖-米尔福德公路，经 Eglinton Valley、镜湖、猴溪，到峡湾后乘双体船游览。

B线：皇后镇飞机直飞米尔福德，抵达后乘船看峡湾，再飞回或车回皇后镇。晚住皇后镇同一家酒店。

7/11TSS蒸汽船 + 瓦尔特峰牧场 + Skyline -> 晚机返程

安排：上午/中午 TSS Earnslaw + Walter Peak，下午 Skyline 缆车，晚上按两人不同航班分别送机。

吃/休息：酒店寄存行李；若航班早于20:00，TSS和Skyline需要二选一或改顺序。

  • 7/6住基督城，7/7出发去特卡波，路线会顺很多。
  • 仍建议7/7-7/9包车，因为特卡波、普卡基湖、库克山和沿途休息点需要灵活停车。

和旅行团确认

  • 7月6日两人航班不同，接机是否能分开安排，是否额外收费。
  • 7月7-9日是否能按基督城-特卡波-库克山-皇后镇主线包车，不安排购物店。
  • 7月11日不包车时，皇后镇酒店寄存行李和分批送机如何安排。
  • 可选景点能加哪些，哪些会明显增加车程或费用。
  • 是否含 Skyline、TSS 蒸汽船、瓦尔特峰农场、米尔福德游船。
  • 7 月是否配雪链，米尔福德封路如何改期或退款。
  • 住宿是否在基督城/特卡波/皇后镇市区或湖边，避免住太偏。
  • 报价是否含接送机、司机小费、早餐、正餐和行李服务。

领导团热门活动推荐

建议锁定

  • 米尔福德峡湾 Coach + Cruise：南岛顶级项目，7/10固定安排。
  • TSS Earnslaw + Walter Peak Farm：7/11安排，船、湖景、农场一次解决。
  • Skyline Queenstown 缆车：7/11下午或傍晚，性价比高，天气好很值。
  • 特卡波观星：7/8晚上看天气，建议选可取消或有室内替代的产品。

预算够可加

  • International Antarctic Centre：7/6航班空档可加，可轻松看到小蓝企鹅救护展示。
  • Onsen Hot Pools：冬天很火，但本次时间紧，只适合7/11航班很晚时加。
  • Gibbston Valley 酒庄：适合拍照、轻松用餐或品酒，但需牺牲7/11部分项目。
  • 皇后镇湖边正式晚餐：比刺激项目更适合商务/领导出行氛围。

这次谨慎

  • Shotover Jet、蹦极、高空跳伞：太刺激，不适合这次领导团。
  • 滑雪：除非大家明确想滑，否则 7 月新手摔倒风险和装备折腾都偏高。
  • 直升机/冰川飞行：很震撼但贵，且 7 月天气取消风险高。
  • 奥马鲁小蓝企鹅：很可爱，但夜间观看会大幅改变路线，本次不建议硬塞。
  • 西海岸冰川、但尼丁、瓦纳卡深度：都好，但7月6-11日时间不够。
